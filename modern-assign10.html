HTML v/s HTML5

HTML HTML5
Its required the use of a flash player to support audio and video It supports audio and video controls with the use of audio and video tags.
It does not allow JavaScript to run in the browser. HTML5 allows JavaScript to run in the background due to JS Web worker API.
HTML uses cookies to store temporary data. HTML5 uses SQL databases and application cache to store offline data.
Older versions of HTML are less mobile-friendly. HTML5 language is more mobile-friendly.
Character encoding in HTML is long and complicated. Character encoding in HTML5 is simple and easy.
